# StudyFlow

## My Project Description
StudyFlow is a web app that helps students organize assignments, track deadlines, and manage study tasks in one place. The goal of the app is to make schoolwork easier to manage by providing a simple and organized planner interface where users can quickly add and track tasks.

## Planned Features
- Add assignments and study tasks with details such as class and due date
- Display assignments in a list so users can easily see upcoming work
- Mark assignments as completed
- Highlight tasks that are due soon or overdue
- Save assignments using local storage so data stays after refreshing the page
